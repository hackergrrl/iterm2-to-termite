# iterm2-to-termite

## Install

```
$ npm install --global iterm2-to-termite
```

## Usage

The program writes the Termite colours to standard out.

```
$ iterm2-to-termite soft-era.itermcolors
color0 = #1c1b28
color1 = #e46b85
color10 = #87b9ac
color11 = #fea3aa
color12 = #70a4dc
color13 = #e897b1
color14 = #a59bd8
color15 = #f8f3f3
color2 = #87b9ac
color3 = #fea3aa
color4 = #70a4dc
color5 = #e897b1
color6 = #a59bd8
color7 = #f8f3f3
color8 = #1c1b28
color9 = #e46b85
background = #f8f3f3
cursor = #e897b1
foreground = #bca3a4
highlight = #e8e4f9
```
